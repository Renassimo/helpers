# Helpers

My various helpers.

## 5book

This is a digital version of my 5book - the book where I every day answer different question, and have opportunity to see my previous answers from the past.

## Avia Helper

This app helps me to create an Instagram hashtags and description from various aircraft data which I spotted.

## Motivation Test

Explore your mind to understand what motivates you at work.

## Game Maps

Track hidden items collection in any open world game.

## My Flight

Here I track all my flights.

## Getting Started

First, install all dependencies:

```bash
yarn
```

Then, run dev server:

```bash
yarn dev
```
